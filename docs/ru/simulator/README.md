# Simulator (RU)

Входная точка для документации приложения-симулятора поведения пользователей GEO.

## Состав папки

- [api.md](api.md)
	- Источник правды по контракту `/api/v1/simulator/*` (snapshot + events + `viz_*`).

- specs/
	- Длинные спецификации уровня продукта/системы.
	- Основной документ: [specs/GENERAL-simulator-application.md](specs/GENERAL-simulator-application.md)
	- Быстрая демка для утверждения визуала (на фикстурах): [specs/10-visual-demo-fast-mock.md](specs/10-visual-demo-fast-mock.md)
	- Прочие спеки/главы: [specs/09-behavior-simulator-application.md](specs/09-behavior-simulator-application.md)

- frontend/
	- Спеки и материалы по UI (игровой интерфейс) и прототипам экранов.
	- Игровой UI (актуальная спецификация): [frontend/Игровой интерфейс симулятора GEO.md](frontend/%D0%98%D0%B3%D1%80%D0%BE%D0%B2%D0%BE%D0%B9%20%D0%B8%D0%BD%D1%82%D0%B5%D1%80%D1%84%D0%B5%D0%B9%D1%81%20%D1%81%D0%B8%D0%BC%D1%83%D0%BB%D1%8F%D1%82%D0%BE%D1%80%D0%B0%20GEO.md)
	- Фаза 1 (tech spec): [frontend/geo-simulator-phase1-tech-spec.md](frontend/geo-simulator-phase1-tech-spec.md)
	- Прототипы экранов: `frontend/screen-prototypes/`
	- Архив: `frontend/archive/`

## Навигация

- Спецификации: `docs/ru/simulator/specs/*`
- Frontend (игровой интерфейс): `docs/ru/simulator/frontend/*`
