# Admin fixtures pack (v1)

Этот каталог содержит **каноничный набор фикстур и сценариев** для прототипирования админки (вариант B: SPA + моки).

- Версия пакета: `admin-fixtures/v1`
- Основная спецификация (RU): см. `docs/ru/admin/specs/admin-ui-prototype-fixtures-spec.md`

Задумка: любой UI (разные агенты/команды) использует **одни и те же** datasets/scenarios → сравнение UX и архитектуры честное.

## Source of truth

- **Canonical fixtures** for the Admin UI demo are generated by the seed generators in `admin-fixtures/tools/` (e.g. Greenfield 100 / Riverside 50).
- `admin-fixtures/tools/generate_admin_fixtures.py` is an **auxiliary** deterministic generator for pagination/edge-case-heavy UI prototyping. It is not the main “community seed”.

## Workflow

1) Generate canonical fixtures (pick one seed):
- `D:/www/Projects/2025/GEOv0-PROJECT/.venv/Scripts/python.exe admin-fixtures/tools/generate_seed_greenfield_village_100.py`
